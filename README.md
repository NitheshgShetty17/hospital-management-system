# Hospital Management System

This project is created as part of the Software Project Planning and Management assignment.

## Project Objective
To design and manage a Hospital Management System that handles patient registration,
doctor details, appointment scheduling, and billing.

## Tools Used
- Jira for task and issue tracking
- Confluence for project documentation
- GitHub for version control

## SDLC Phases Covered
- Requirement Analysis
- System Design
- Development
- Testing
- Documentation
